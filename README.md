# Object Detector
This is for a school project, in which I will be using Arduino, and the language C++ for the code. Millions on people on this planet suffer with Blindness and can't see what is in front of them, through this device they will be able to know if things are in front of them.
## How does this work?
As I previously mentioned, I will be using a Arduino Uno, breadboard, sensors and a buzzer to make this. I will be programming the Arduino Board using C++ which is a successor to C. This will be achieved through Arduino IDE. If you want the code, you can check the other files in this repository.
